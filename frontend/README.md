# SciGrader - 理工科 AI 作业批改系统

## 📚 项目简介

SciGrader 是一个现代化的教育类后台系统，专为理工科教学设计，提供 AI 辅助的作业批改功能。系统采用简洁的卡片式设计和仪表盘布局，界面清晰、易用性强。

## ✨ 主要功能

### 1. 登录界面
- **角色切换**：支持教师端和学生端一键切换
- **演示账号**：
  - 教师端：`admin` / `123456`
  - 学生端：`student1` / `123456`
- **安全验证**：密码加密存储和验证

### 2. 教师端功能
#### 题目管理
- **新建题目**：
  - 支持三种题型：概念题、计算题、编程题
  - 三个难度等级：简单、中等、困难
  - 自定义题目编号和分值
  - 支持公式、代码等多行文本输入
  - 知识点标签管理
  - 附件上传（PDF、PNG、JPG、TXT、代码文件）

- **已发布作业**：
  - 查看所有已发布的作业
  - 显示作业编号、标题、发布时间、截止时间
  - 查看学生提交情况
  - 实时状态监控（进行中/已截止）

- **批改后作业**：
  - 查看系统自动批改结果
  - 查看学生答案和系统评分
  - 查看系统评语
  - 批改时间记录

#### 快捷操作
- 批量导入题目（Excel/Word）
- 知识点标签管理
- AI 难度评估

### 3. 学生端功能
#### 我的作业
- **作业列表**：
  - 显示所有已发布的作业
  - 三种状态标识：
    - 🟡 待提交（黄色）
    - 🟢 已提交（绿色）
    - 🔴 已逾期（红色）
  
- **提交作业**：
  - 在线作答（支持文本、公式、代码）
  - 附件上传
  - 自动判断提交状态（正常/逾期）

- **查看提交**：
  - 查看已提交的作业内容
  - 查看提交时间和状态

#### 批改后作业
- 查看已批改的作业详情
- 查看系统评分和评语
- 查看教师评语（如有）
- 批改时间记录

#### 学习统计
- **数据统计面板**：
  - 总作业数
  - 已完成作业数
  - 待提交作业数
  - 已逾期作业数
  - 平均得分

- **框架**：Streamlit 1.32.0
- **数据库连接**：mysql-connector-python 8.3.0
- **数据处理**：pandas 2.1.4

### 后端数据库
- **数据库**：MySQL 8.0+
- **字符集**：utf8mb4_unicode_ci
- **存储引擎**：InnoDB

## 📦 安装与部署

### 环境要求
- Python 3.8 或更高版本
- MySQL 8.0 或更高版本
- 现代浏览器（Chrome、Edge、Firefox）

### 安装步骤

#### 1. 数据库配置
```bash
# 登录 MySQL
mysql -u root -p

# 执行数据库脚本
source database/scigrader_schema.sql
```

#### 2. 安装依赖
```bash
cd frontend
pip install -r requirements.txt
```

#### 3. 配置数据库连接
编辑 `frontend/.streamlit/secrets.toml`：
```toml
[database]
host = "localhost"
port = 3306
user = "root"
password = "你的密码"
database = "scigrader_db"
```

#### 4. 启动应用

**Windows：**
```bash
cd frontend
start.bat
```

**Linux/Mac：**
```bash
cd frontend
streamlit run app.py --server.port=8501
```

### 访问地址
- 本地访问：http://localhost:8501
- 局域网访问：http://你的 IP:8501

## 📊 数据库设计

### 核心数据表

1. **users** - 用户表
   - 支持教师和学生两种角色
   - 包含用户名、密码哈希、邮箱等

2. **problems** - 题目表
   - 题目类型、难度、内容
   - 参考答案、知识点标签
   - 附件信息

3. **assignments** - 作业表
   - 作业编号、标题、说明
   - 课程名称、发布时间、截止时间
   - 关联题目列表

4. **submissions** - 提交记录表
   - 学生答案、提交文件
   - 自动评分、教师评分
   - 批改评语

5. **grading_records** - 批改记录表
   - 详细批改信息
   - 批改人和批改时间

6. **knowledge_points** - 知识点表
   - 知识点名称和描述
   - 支持层级关系

7. **learning_statistics** - 学习统计表
   - 学生作业完成情况统计
   - 平均分计算

### 数据关联

- **教师创建题目** → **problems.creator_id** → **users.user_id**
- **教师发布作业** → **assignments.teacher_id** → **users.user_id**
- **学生提交作业** → **submissions.student_id** → **users.user_id**
- **作业包含题目** → **assignments.problem_ids** → **problems.problem_id**
- **批改记录** → **grading_records.submission_id** → **submissions.submission_id**

### 视图设计

1. **assignment_submission_view** - 作业提交详情视图
   - 联合作业表和提交表
   - 方便双端查询显示

2. **student_statistics_view** - 学生学习统计视图
   - 自动计算各项统计数据
   - 实时更新

## 🔐 安全说明

### 密码安全
- 当前使用 SHA-256 哈希（演示用途）
- **生产环境请使用 bcrypt 或 argon2**

### 会话管理
- Streamlit session_state 管理登录状态
- 退出登录时清除所有会话数据

### 数据库安全
- 使用参数化查询防止 SQL 注入
- 数据库连接信息存储在 secrets.toml（不提交到版本控制）

## 📝 使用说明

### 教师操作流程

1. **登录系统**
   - 选择"教师端"
   - 输入账号密码

2. **创建题目**
   - 填写题目编号、类型、难度
   - 输入题目内容和参考答案
   - 选择知识点标签
   - 上传附件（可选）
   - 点击"创建题目"

3. **发布作业**
   - 在已发布作业模块创建新作业
   - 选择题目组成作业
   - 设置截止时间
   - 发布作业

4. **查看提交**
   - 点击"查看提交情况"
   - 查看学生答案
   - 查看系统评分

### 学生操作流程

1. **登录系统**
   - 选择"学生端"
   - 输入账号密码

2. **查看作业**
   - 在"我的作业"查看作业列表
   - 注意截止时间和状态

3. **提交作业**
   - 点击"提交作业"
   - 输入答案或上传文件
   - 确认提交

4. **查看批改**
   - 在"批改后作业"查看详情
   - 查看评分和评语
   - 查看学习统计
